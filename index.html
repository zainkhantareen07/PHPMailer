üöÄ Installation & Setup

1. Download

Clone or download this repository to your local machine.

git clone https://github.com/zainkhantareen07/PHPMailer.git

2. Web Server

Place files in your web server's document root (htdocs/www folder).

Requires: Apache/Nginx with PHP 7.0+

3. Dependencies

Install PHPMailer via Composer (optional, has fallback).

composer install

4. Configuration

Configure SMTP settings in phpmailer_config.php.

Copy from phpmailer_config.php.example

SMTP Configuration Example:

'smtp.gmail.com', 'username' => 'your-email@gmail.com', 'password' => 'your-app-password', 'smtp_secure' => 'tls', 'port' => 587, 'smtp_auth' => true, 'smtp_debug' => 0 ]; ?>
